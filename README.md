## 📌 About
This project was developed during the **Alura – Clustering: Working with Unlabeled Data** course, with the goal of applying unsupervised learning techniques for user segmentation.

The application uses the **K-Means clustering algorithm** to identify patterns in unlabeled data and group users according to their interests. This approach makes it possible to generate strategic insights for more targeted and effective marketing campaigns.

---

## ⚙️ How the Project Works
The project follows the workflow below:

1. Upload a CSV file containing user data
2. Encode categorical variables (such as gender)
3. Normalize the data
4. Apply the trained K-Means model
5. Classify users into interest groups
6. Download the results with the assigned clusters

The application uses pre-trained models loaded with **joblib** to perform predictions.
