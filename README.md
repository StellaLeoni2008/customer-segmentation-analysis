## 📌 Sobre
Este projeto foi desenvolvido durante o curso **Clusterização: lidando com dados sem rótulo da Alura**, com o objetivo de aplicar técnicas de aprendizado não supervisionado para segmentação de usuários.
A aplicação utiliza o algoritmo K-Means para identificar padrões em dados sem rótulo e agrupar usuários com base em seus interesses. Com isso, é possível gerar insights estratégicos para campanhas de marketing mais direcionadas e eficientes.


## ⚙️ Como funciona o projeto
O fluxo do projeto segue as seguintes etapas:
  1. Upload de um arquivo CSV contendo dados dos usuários
  2. Codificação de variáveis categóricas (como sexo)
  3. Normalização dos dados
  4. Aplicação do modelo K-Means treinado
  5. Classificação dos usuários em grupos de interesse
  6. Download do resultado com os grupos atribuídos
O modelo utiliza arquivos previamente treinados, carregados com joblib para realizar as previsões.
